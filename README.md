# vue-vux-iconan
vue-vux搭建的webapp和混合应用

``` bash
# 初始化
npm install

# 开发模式，端口7070
gulp dev

# 在安卓手机上调试（插好USB线）
gulp run.android

# 打包webapp
gulp build
gulp build -t webapp

# 打包安卓apk
gulp build -t mobile
```
